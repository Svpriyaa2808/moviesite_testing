welcome to POPCORN PICKS site

        Our website Popcorn picks is about movies.We have four pages included HOME,FILM FOCUS,MY PICKS and POPCORN FACTS.
    We have data for nearly 18 movies which consists of Movie name , Released year , Genre of movie and description about the movie .
    The website has filter by genre button to display some genre of movie, Back button is used to display all movie.
    In addition to this the Favourite icon is displayed, when user click the favourite icon that particular movie will add to local storage and 
    displayed under My picks category.



HOME PAGE:

        In home page content some basic content related to movie is displayed with header at top containing <h1>tag and logo and Footer at the bottom with
    links to social media pages.

FILM FOCUS:

        In Film focus page ,on page load all movies in the dataset are displayed .Each movie has icon , Moviename , Release year , Genre and Favourite icon.
    When the user clicks on the moviename of any movie , the description of that movie is displayed in popup with close button.If the close button is 
    clicked, the description container will goes off and all movies displayed.When the mouse enter the movie icon it will rotate 360deg and its alt name 
    changed to moviename rotated and when it leaves it rotates back to original and alt will change to moviename not-rotated.

    FILTER BUTTON FUNCTION: 

        When filter by genre button is clicked,Four types of genre (Horror,Romantic,Actio,Comedy) is displayed and again the button is clicked,the genres
        list will disappears. If user clicks on Horror , list of Horror movies will be displayed.


    BACK BUTTON FUNCTION:

        If list of horror movies is displayed and back button is clicked ,then list of all movies displayed.

    FAVOURITE ICON FUNCTION:

        Each movie has favourite icon which is Black heart icon , if user clicks on favourite icon the black heart icon changes into red heart icon
        and that movie will be displayed under My picks category
